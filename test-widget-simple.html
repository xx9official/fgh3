ZYMO Chat Widget - Enhanced Test

This is an enhanced test page to demonstrate the ZYMO chat widget with problem description, waiting status, and supporter claiming features.

New Features to Test:

  • Problem description form when you first open the chat
  • Waiting status while supporter claims the chat
  • Supporter profile and online/offline status

  • Support link for illegal activities reporting

Test Flow:

  1. Click the chat button
  2. Fill out the problem description form
  3. See the waiting status
  4. Wait for "supporter" to claim the chat (simulated)
  5. Chat with the supporter

Note: This is a demo widget. The supporter claiming is simulated and will happen automatically after 3 seconds.
